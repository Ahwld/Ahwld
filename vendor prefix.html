# MORORO Config files for my GitHub profile. Document

벤더 프리픽스(VENDOR PREFIX) 소개


  • 벤더 프리픽스 : 주요 웹 브라우저 공급자가 새로운 실험적인 기능을 제공할 때
    이전 버전의 웹 브라우저에 그 사실을 알려주기 위해 사용하는 접두사(prefix)를 의미한다.

  • 아직 CSS 권고안에 포함이 되지 못한 기능이나,
    CSS 권고안에는 포함되어 있지만 아직 완벽하게 제정된 상태가 아닌 기능을 사용하고자 할 때 벤더 프리픽스를 사용한다.
    그렇게 하면 해당 기능이 포함되어 있지 않은 이전 버전의 웹 브라우저에서도 그 기능을 사용할 수 있다.
